# Pricing and paid options

This repository is the free public preview. It is intentionally useful on its own, but the paid offers are for teams that want more automation coverage or implementation help.

## Draft paid options

| Option | Draft price | Includes | Best for |
|---|---:|---|---|
| Free Lite | $0 | GitHub weekly snapshot workflow | Trying the workflow style before sharing credentials with any tool |
| Starter template pack | $19 | Full importable workflow pack, sample payloads/outputs, setup and safety docs | Operators who can self-host n8n and wire their own credentials |
| Pro template pack | $49 | Starter + customization worksheets, demo/runbook, risk-demo table, onboarding notes | Consultants and teams adapting workflows repeatedly |
| AI Workflow Audit beta | $300–700 | 3-business-day workflow audit, ranked backlog, ROI/security scorecard, one implementation spec | Teams that need help deciding what to automate safely |
| 2-week implementation pilot | Scoped after audit | One narrow workflow shipped in draft/review mode | Teams ready to connect approved tools and owners |

## Important boundaries

- No guaranteed ROI claims.
- No production credentials are requested through GitHub issues.
- No private customer data should be posted publicly.
- Paid implementation, invoicing, tax, KYC, and contract details are handled only after private scope approval.
- Initial GitHub issues should contain sanitized context only.

For a public-safe explanation of what to share before any paid/private scope exists, see `docs/upgrade-path-boundary.md`. For a checkout-disabled public listing preview of the broader pack, see `docs/full-pack-public-listing.md`. To decide whether an inquiry is a fit before posting public details, use `docs/buyer-fit-checklist.md`.

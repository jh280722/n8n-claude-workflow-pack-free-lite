# Workflow landing guide

Use this page if you landed directly in the `workflows/` folder or opened `free-lite-github-weekly-snapshot.json` from GitHub traffic/search instead of starting from the repository README.

## Safe first path

1. Download or copy `workflows/free-lite-github-weekly-snapshot.json` only from this public repository.
2. Import it into your own n8n instance.
3. For the first run, use a public repository such as `n8n-io/n8n` and leave `GITHUB_TOKEN` blank.
4. Inspect the final `markdown` field before adapting the workflow for Slack, Notion, Discord, Claude, or private repositories.
5. If you need step-by-step setup, follow `../docs/free-lite-download-first-run-guide.md` and `../docs/free-lite-import-checklist.md`.

## What not to share publicly

Keep public issues and screenshots public-only. Do not post tokens, credentials, private repository URLs, customer data, production logs, checkout/payment details, payout/wallet/bank/Stripe details, tax/KYC/contract details, DM/email/forms/private outreach requests, paid ads instructions, or guaranteed ROI assumptions.
